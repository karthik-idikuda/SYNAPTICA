# Synaptica

Voice-activated personal assistant for workflow automation with task scheduling, reminders, and third-party app integrations.

## Features

- Voice command recognition and processing
- Task scheduling and reminder management
- Third-party app integrations (calendar, email, messaging)
- Customizable automation workflows

## Tech Stack

Python, Speech Recognition, NLP, REST APIs

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/SYNAPTICA.git
cd SYNAPTICA
pip install -r requirements.txt
```

### Usage

```bash
python assistant.py
```
